开发日志：搭建自己的小站之路(1)

上礼拜周六凌晨，大概1点钟的样子，就辣么一瞬间，终于拿起了自己设想已久--要给自己建个小站--的想法。俗话说：万事开头难，事情起步了，总会有那么点动力会驱使自己慢慢去完成它。当初的想法很让人激动，但是过去很久时间，却一直没踏出第一步。于是，那晚，2015-11-06日，终于狠下心买了云服务器，开始了行动路上的第一步。虽然是最低配，装的还是很久未接触的linux，不过还是通宵搭好了环境:jdk，数据库，nginx/tomcat。紧接着，服务器有了，便是开发：搭框架，选java模板(不用jsp)，搜静态页。到了今晚，终于把nginx+tomcat负载均衡完全完成，项目可以静动分离的正常跑起来。虽然现在还是相当于一些网上拿来的静态页面(少量后端动态交互)，但是竟然有点小小的激动,等待哥后续完成~
